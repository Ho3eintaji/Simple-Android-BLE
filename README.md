# BLE Sensor Connector

A simple Android app built with Kotlin and Jetpack Compose for connecting to a sensor via BLE (Bluetooth Low Energy).

## Features
- Connect to BLE sensors
- Easy-to-use Jetpack Compose UI
- Lightweight and efficient

## Technologies Used
- Kotlin
- Jetpack Compose
- Bluetooth Low Energy (BLE)
